# Python Lexicon Course

## Description
This repository contains my work for the Python Lexicon course.
It includes exercises, assignments, and small projects created to
practice core Python concepts such as variables, control flow,
functions, object-oriented programming, and basic project structure.

The goal of this repository is to document my learning progress and
showcase my understanding of Python fundamentals.
